# User Journey: Creación de Memoria

## Objetivo

Crear un archivo Markdown dedicado (`USER_JOURNEY.md`) con un user journey empático y simple que guíe al usuario paso a paso en la creación de una memoria.

## Estructura del Archivo

El archivo incluirá:

1. **Introducción empática**: Contexto sobre la importancia de preservar recuerdos
2. **5 pasos detallados**:

- Paso 1: Conocer a la Persona (nombre, relación, descripción)
- Paso 2: Capturar su Esencia Visual (subida de foto)
- Paso 3: Preservar su Voz (subida de audio para clonación)
- Paso 4: Compartir sus Historias (memorias favoritas y rasgos de personalidad)
- Paso 5: Personalización Final (revisión y confirmación)

3. **Elementos en cada paso**:

- Mensaje de bienvenida empático
- Campos a completar
- Ejemplos y guías
- Validaciones y retroalimentación
- Transición al siguiente paso

## Archivos a Crear

- `USER_JOURNEY.md`: Archivo nuevo en la raíz del proyecto con el user journey completo
